# CIPHER NEXUS 项目重构总结

## 📁 项目架构

### 模块化目录结构
```
cipher-nexus/
├── index.html          # 主 HTML 文件（已精简，只保留结构和样式配置）
├── src/
│   ├── css/
│   │   └── styles.css  # 自定义样式（包含所有 CSS 动画和效果）
│   └── js/
│       ├── state.js    # 状态管理模块
│       ├── ciphers.js  # 密码算法库（17 种算法）
│       ├── views.js    # 视图管理模块
│       ├── history.js  # 历史记录管理模块
│       ├── ui.js       # UI 工具模块
│       ├── cipher-ui.js # 密码选择 UI 模块
│       ├── core.js     # 核心加密/解密执行模块
│       ├── batch.js    # 批量处理模块（新增）
│       ├── analyzer.js # 密文强度分析模块（新增）
│       └── main.js     # 主应用入口
├── assets/             # 资源目录（预留）
├── README.md
└── LICENSE
```

## 🔧 重构亮点

### 1. 模块化架构
- **HTML/CSS/JS 完全分离**：从 1322 行单文件拆分为清晰的多模块结构
- **职责分离**：每个模块专注于单一功能，便于维护和扩展
- **全局作用域管理**：通过模块化设计减少全局变量污染

### 2. 核心模块设计

#### 状态管理 (state.js)
- 统一的AppState 对象管理全局状态
- LocalStorage 持久化存储
- 自动保存和加载历史记录
- 操作统计追踪

#### 密码算法库 (ciphers.js)
- **17 种密码算法**：
  - 古典密码：摩斯密码、凯撒密码、ROT13、埃特巴什、栅栏密码、维吉尼亚密码、仿射密码、培根密码、普莱费尔密码（新增）
  - 现代编码：Base64、二进制、十六进制、URL 编码、ASCII 码、反转
  - 哈希算法：MD5、SHA-1、SHA-256、SHA-512（新增）
- 统一的算法接口（encrypt/decrypt）
- 参数化配置支持

#### 视图管理 (views.js)
- 视图切换和状态管理
- 动态渲染密码选择网格
- 参数配置区域动态生成
- 密码图鉴渲染

#### 历史记录 (history.js)
- 历史记录渲染和管理
- 批量导出功能
- 记录恢复和重用

#### UI 工具 (ui.js)
- Toast 提示系统（带动画）
- 文件上传/下载
- 剪贴板操作
- 字符计数
- 加载遮罩

#### 核心操作 (core.js)
- 加密/解密执行引擎
- 参数传递
- 历史记录保存
- 性能计时

#### 批量处理 (batch.js) - 新增
- 批量文本处理
- 进度显示
- CSV 导出
- 错误处理

#### 密文分析 (analyzer.js) - 新增

- 字符分布分析
- 强度评分系统
- 改进建议生成

## 🎯 核心机制

### 加密流程
```
用户输入 → 选择算法 → 配置参数 → 执行加密 → 显示结果 → 保存历史 → 更新统计
```

### 解密流程
```
输入密文 → 选择/识别算法 → 配置参数 → 执行解密 → 显示明文 → 保存历史
```

### 批量处理流程
```
批量输入 → 解析文本行 → 逐项处理 → 进度更新 → 结果汇总 → 导出 CSV
```

### 密文分析流程
```
获取输出 → 计算熵值 → 分析分布 → 综合评分 → 生成建议 → 可视化展示
```

## 📝 使用指南

### 基本使用
1. 打开 `index.html`（或访问部署的网页）
2. 选择加密/解密模式
3. 选择密码算法
4. 输入文本
5. 配置参数（如需要）
6. 执行操作

### 批量处理
1. 点击"批量"按钮
2. 在批量输入区输入多行文本
3. 点击"开始批量处理"
4. 等待处理完成
5. 导出 CSV 结果

### 密文分析
1. 执行加密/解密操作
2. 点击"分析"按钮
3. 查看强度分析报告
