### ✅ 2. 组合密码功能 (combo.js)

**支持 2-3 重密码组合加密/解密：**

#### 预设组合方案（6 种）
1. **双重凯撒** - 凯撒密码加密两次，使用不同偏移量
2. **凯撒 + 反转** - 先凯撒加密，再反转字符串
3. **三重混合** - 凯撒 + 反转 + 十六进制
4. **维吉尼亚 + 凯撒** - 先维吉尼亚，再凯撒
5. **摩斯 + 二进制** - 先摩斯密码，再二进制
6. **Base64 + 反转** - 先 Base64，再反转

#### 核心特性
- 支持自定义参数配置
- 自动按顺序应用多个密码
- 解密时自动逆序处理
- 动态参数配置界面
- 结果保存到历史记录

#### 技术实现
```javascript
// 加密流程
text → Cipher1 → Cipher2 → Cipher3 → result

// 解密流程
result → Cipher3⁻¹ → Cipher2⁻¹ → Cipher1⁻¹ → text
```

### ✅ 3. 多重翻译功能 (multi-translate.js)

**同时翻译为多种格式：**

#### 预设翻译方案（4 种）
1. **古典三重奏** - 摩斯 + 二进制 + 十六进制
2. **现代三重奏** - Base64 + URL + ASCII
3. **混合四重奏** - 反转 + 凯撒 + 二进制 + 十六进制
4. **自定义** - 自由选择最多 5 种密码

#### 核心特性
- 一键生成多种格式结果
- 支持自定义组合（最多 5 种）
- 每个结果独立显示，可单独复制
- 支持导出为 JSON 或文本格式
- 错误处理和状态显示

#### 使用场景
- 对比不同编码格式
- 快速生成多种密文
- 教学演示多种编码
- 数据多格式备份
