## 🔥 新功能使用

### 1. 组合密码（推荐⭐）

**什么是组合密码？**
将 2-3 种密码组合使用，提供更高级别的安全性。

**使用步骤：**
1. 点击顶部导航「组合密码」
2. 选择预设方案（如：双重凯撒）
3. 配置参数（如需要）
4. 输入文本
5. 点击「组合加密」或「组合解密」

**推荐组合：**
- 🔐 **双重凯撒** - 适合初学者
- 🔐 **凯撒 + 反转** - 简单有效
- 🔐 **三重混合** - 高安全性
- 🔐 **摩斯 + 二进制** - 教学演示

**示例：**
```
原文：HELLO
组合：双重凯撒 (shift1=3, shift2=7)

加密过程:
HELLO → (凯撒 1) → KHOOR → (凯撒 2) → RUVVY

最终输出：RUVVY
```

---

### 2. 多重翻译（推荐⭐）

**什么是多重翻译？**
一键将文本同时翻译为多种格式。

**使用步骤：**
1. 点击顶部导航「多重翻译」
2. 选择预设方案（如：古典三重奏）
3. 输入文本
4. 点击「执行多重翻译」
5. 查看多种格式的结果

**预设方案：**
- 📜 **古典三重奏** - 摩斯 + 二进制 + 十六进制
- 📱 **现代三重奏** - Base64 + URL + ASCII
- 🎲 **混合四重奏** - 反转 + 凯撒 + 二进制 + 十六进制
- 🛠️ **自定义** - 自由选择最多 5 种

**示例：**
```
原文：HELLO

选择「古典三重奏」后输出:

【摩斯密码】
.... . .-.. .-.. ---

【二进制】
01001000 01000101 01001100 01001100 01001111

【十六进制】
48 45 4c 4c 4f
```

## 💪 高级技巧

### 批量处理技巧

**使用场景：** 需要加密/解密多段文本

**步骤：**
1. 在工作台点击「批量」按钮
2. 在批量输入区输入多行文本（每行一个）
3. 点击「开始批量处理」
4. 等待处理完成
5. 点击「导出 CSV」保存结果

**示例：**
```
输入:
Hello
World
Test

输出:
[1] Khoor
[2] Zruog
[3] Whvw
```

---

### 密文强度分析

**使用场景：** 评估加密后的安全性

**步骤：**
1. 执行加密操作

**分析指标：**
- 📏 **长度** - 文本长度

- ⭐ **唯一字符** - 多样性
- 📊 **综合评分** - 0-100 分
- 💡 **改进建议** - 优化方案

## 🛠️ 故障排除

### 问题 1: 页面无法加载
**解决方案：**
- 检查浏览器是否支持 ES6
- 清除浏览器缓存
- 使用 Chrome/Firefox/Edge 最新版

### 问题 2: 加密结果不正确
**解决方案：**
- 检查参数配置是否正确
- 确认选择的密码算法
- 检查输入文本格式

### 问题 3: 快捷键不生效
**解决方案：**
- 确保在工作台视图
- 检查是否与其他快捷键冲突
- 按 F1 查看快捷键列表

### 问题 4: 设置无法保存
**解决方案：**
- 检查浏览器是否允许 LocalStorage
- 清除浏览器数据后重试
- 尝试其他浏览器
